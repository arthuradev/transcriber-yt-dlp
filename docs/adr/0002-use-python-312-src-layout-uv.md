# ADR — Use Python 3.12, src layout, and uv

## Status
Accepted

## Context
The project needs modern Python, reproducible dependency management, and clean package layout.

## Decision
Use Python 3.12+, `src/` layout, and uv for dependency management.

## Consequences
Better tooling and packaging; contributors need uv installed.

## Notes for AI agents
Do not reverse this decision without creating a new ADR and explicit user approval.
