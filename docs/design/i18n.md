# Internationalization

## Languages
- Portuguese (`pt-BR`).
- English (`en-US`).

## First run
The app asks:

```text
Choose your language / Escolha seu idioma:
- Português
- English
```

The setting is stored locally and can be changed later.

## Rules
- Code identifiers remain English.
- Public README is bilingual.
- User-facing strings go through the language system.
- Do not hardcode text in business logic.
