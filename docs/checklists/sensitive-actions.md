# Sensitive Actions Checklist

Before executing a sensitive action:

- [ ] Structured command exists.
- [ ] Risk level classified.
- [ ] Dry-run shown.
- [ ] Preconditions validated.
- [ ] User confirmation obtained if required.
- [ ] Secrets redaction enabled.
- [ ] Rollback/cleanup considered.
- [ ] Audit event emitted.
- [ ] Final report generated.
