# Roadmap

Each phase has an associated tag and GitHub Release.

| Phase | Tag | Title | Goal |
|---:|---|---|---|
| 1 | `v0.1.0` | Foundation, docs, repo, CI, architecture | Create public repo, docs, skeleton, quality gates |
| 2 | `v0.2.0` | UI base | Rich/Questionary/Pyfiglet startup and menu |
| 3 | `v0.3.0` | Themes, ASCII art, clean screen | Theme system and success art renderer |
| 4 | `v0.4.0` | Config, language, first-run setup | User settings, i18n, first-run flow |
| 5 | `v0.5.0` | WeatherAPI and personalization | Optional weather/time display |
| 6 | `v0.6.0` | yt-dlp metadata probe | Probe URLs and show metadata |
| 7 | `v0.7.0` | Download planner + dry-run | Create safe operation plans |
| 8 | `v0.8.0` | Download executor + progress | Execute downloads with progress |
| 9 | `v0.9.0` | Profiles and formats | Quality/profile selection |
| 10 | `v0.10.0` | Playlists, batch URLs, folders, duplicates | Batch/playlist support and organization |
| 11 | `v0.11.0` | GPU-only transcription | faster-whisper CUDA workflow |
| 12 | `v0.12.0` | Subtitles/transcripts | Download existing subtitles/transcripts |
| 13 | `v0.13.0` | DeepSeek/OpenAI-compatible cleanup | Optional transcript formatting |
| 14 | `v0.14.0` | Safety pipeline and cookies | Full safety model and cookie guard |
| 15 | `v0.15.0` | History, logs, reports | SQLite/history and report output |
| 16 | `v0.16.0` | Complete tests and architecture checks | Coverage target and hard gates |
| 17 | `v0.17.0` | UI/UX polish | Animations, themes, messages, layouts |
| 18 | `v0.18.0` | Windows bootstrap | bootstrap.ps1 and dependency checks |
| 19 | `v0.19.0` | Packaging | PyInstaller portable exe and installer plan |
| 20 | `v0.20.0` | Public release candidate | Final docs, release polish, public usability |

`v1.0.0` is intentionally not part of this roadmap. It requires explicit user approval.
