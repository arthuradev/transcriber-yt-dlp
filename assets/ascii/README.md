# ASCII Assets

This folder stores UTF-8 ASCII/Unicode art used by the TUI.

## Rules
- Preserve leading spaces.
- Do not wrap lines.
- Render only if terminal width is sufficient.
- Use fallback compact message if too small.
- Users can add custom `.txt` files locally.

## Folders
- `welcome/`: optional welcome art.
- `success/`: shown after successful operations.
- `custom/`: user-added local art.
